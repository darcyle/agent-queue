# Feature Requests

## Stop Task Button in Task Start Messages

When tasks are started, there should be a button in the Discord message to stop the task if the user desires. This would provide a quick way for users to cancel a task without needing to use slash commands or find the task ID separately.

Ideally the button should:
- Appear in the message posted when a task is assigned to an agent
- Trigger `stop_task` when clicked
- Provide confirmation feedback when the task is stopped

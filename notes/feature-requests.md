# Feature Requests

### Interactive Menu Message

A persistent status message in the main chat with clickable buttons for common actions:
- **Status overview** (live updating)
- **List active tasks** (per project or all)
- **Open notes**
- **List projects**
- **Add new task**
- **Restart task**
- **List agents**
- **Toggle orchestrator**
- And more...

Button menus would allow remote control without typing commands — useful for mobile/DSM usage.
